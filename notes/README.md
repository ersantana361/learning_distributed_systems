# Personal Notes

This directory contains personal annotations, questions, and insights gathered while studying the Distributed Systems lecture series.

## Structure

- `section_N_*.md` - Notes organized by course section
- `questions.md` - Questions to revisit, topics needing clarification

## How to Use

1. **During lectures**: Jot down questions, confusing points, or "aha" moments
2. **After lectures**: Summarize key takeaways in your own words
3. **Review sessions**: Revisit questions.md and try to answer them
4. **Connections**: Note how concepts relate to real-world systems you've worked with

## Note Template

Each section file follows this format:

```markdown
## Key Concepts
- Concept 1: Brief explanation in your own words
- Concept 2: ...

## Questions
- [ ] Question that came up during lecture
- [x] Question you've resolved (with answer)

## Connections to Real Systems
- How concept X applies to system Y

## Things to Revisit
- Topics needing more study
```
